  • What is front-end development?

    Front-end web development allows you to build beautifully crafted and well coded simple websites.

  • What will I learn to do?

    You will learn how to quickly turn ideas into working prototypes, and be able to design and build a well structured website

  • What are the course requirements?

    This course is for beginners who are eager to learn web development. You will be required to put in a lot of work to get the most of this course. A laptop is also required.

Course curriculum

  1. Introduction
    • Overview of the schedule
    • Setup of development environment
    • High-level introduction to HTML and CSS
    • High-level introduction to Git and Github
    • Brief overview of how the web works (servers, clients, etc.)
  2. HTML
    • Brief history of HTML
    • Introduction to HTML's role, structure vs. presentation
    • Understanding what is meant by HTML5
    • Using HTML semantic tags to create meaningful documents
    • Understanding of images, differences between the various image formats
  3. CSS
    • Introduction to CSS' role, structure vs. presentation
    • Experimenting with basic CSS properties
    • Understanding of the CSS cascade and specificity
    • Understanding of the CSS box model
    • Use a browser's inspector to interact with the content
    • Use media queries to create a responsive experience
  4. JavaScript
    • Explain the use of Javascript in front-end development
    • High-level understanding of imperative vs. declarative coding
    • Javascript language: variables, data types, control flow, functions
    • Introduction to the DOM and its manipulation
    • Using jQuery for smoother DOM manipulations
  5. Responsive Framework: Foundation
    • High-level discussion about responsiveness
    • Using our newly-acquired understanding of front-end technologies, explore Foundation’s offerings
    • Demo: creating a small landing page using Foundation’s responsive components
  6. Project Week
    • Using everything that was learned through the sessions, create a personal landing page or a microsite. This project can be done alone or with a group. Your instructor will control the scope of the project, and will be guiding you throughout.
  7. During project week, students will work on their own project under the guidance of their instructor.

FAQ

  • Do I need any prior knowledge to attend this course?

    No, this course welcomes beginners who have never written a line of code before, or those who have tinkered with a minimal amount of code.

  • What are the requirements to take part in this course?

    You will need to be ambitious, focused, willing to work on your own time, and be receptive to learning new things. You will also need to bring a laptop. (Mac Preferred but not necessary)

  • Does everyone get accepted for the course?

    Only those candidates which we feel will benefit the most from this course will be accepted.

  • How many students are in the classroom?

    The course is limited to a maximum of 10 students. Seats tend to fill up quickly so make sure to apply as soon as possible.

  • Is 8 part-time weeks enough to learn how to code?

    Yes and no. It’s an excellent start and you can become a junior front-end web developer. However, you will need to continue learning and practicing to refine and grow your skill set.

  • How much does the course cost?

    Tuition for the course is $2,500 and is payable in two equal payments. We accept all major forms of payment.

  • What are the payment deadlines?

    Once you are accepted into the course, you will be required to pay a $250 deposit to secure your seat. The remaining balance is split into two equal payments. The first due on the starting day of the course, and the second due midway through the course.
